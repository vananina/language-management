# language-manage-site
imez language management system

# project 환경 구성
빌드관리 : webpack ^4 + webpack-cli 3.2.0
주 사용 js library: vue 2.5.21 + @vue/cli($npm install -g @vue/cli 최신버전으로 ^3)
es6 문법 사용 권장 

# step 1
npm install

# production mode (minified)
npm run build

# development mode
npm run dev

# 개발 환경 모드로 로컬에서 웹 실행
npm run start
